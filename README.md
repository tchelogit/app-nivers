# Aniversários App Documentation

## Overview
The Aniversários app is designed to help users manage and celebrate their loved ones' birthdays. It provides an intuitive interface for tracking upcoming birthdays and sending reminders.

## Features
- User-friendly interface for easy navigation.
- Automatic birthday reminders.
- Option to send greetings via various channels.
- Customizable birthday messages.

## Installation Instructions
To build and run the Aniversários app, follow these steps:

1. **Clone the repository**:  
   Clone the repository from GitHub:  
   ```bash  
   git clone https://github.com/tchelogit/app-nivers.git  
   ```  

2. **Navigate to the project directory**:  
   ```bash  
   cd app-nivers  
   ```  

3. **Install dependencies**:  
   Make sure you have [Node.js](https://nodejs.org/) installed, then run:  
   ```bash  
   npm install  
   ```  

4. **Build the project**:  
   To build the project, use the following command:  
   ```bash  
   npm run build  
   ```  

5. **Run the application**:  
   After building the project, you can run the application using:  
   ```bash  
   npm start  
   ```  

## Usage Instructions
- Open the app in your web browser.
- Create an account to start adding important dates.
- Populate the app with birthday entries for your friends and family.
- Set reminders to receive notifications via email or push notifications.
